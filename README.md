# 리팩토링


Extract Method : 코드 덩어리를 별도의 코드로 분리

그룹으로 묶을 수 있는 코드가 있다면 적절한 메소드를 만들어서 해당 코드를 메소드 안으로 이동

Inline Method : 위의 반대 개념
