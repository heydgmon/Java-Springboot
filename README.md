String은 불변적인 특징 때문에 값을 업데이트하면, 매 연산 시마다 새로운 문자열을 가진 String 인스턴스가 생성되어 메모리공간을 차지하므로 연산 속도가 낮음


String buffer는 가변적인 특징 때문에 값을 업데이트할 시 별도의 인스턴스 생성 없이 연산이 이루어지므로 연산 속도가 빠름
